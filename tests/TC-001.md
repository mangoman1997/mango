# TC-001: Normal request with valid signature

- Preconditions:
  - Webhook endpoint available at /webhook/egpay
  - SIGNATURE_SECRET configured
- Steps:
  - Send POST to /webhook/egpay with payload {"event":"pay_success","order_id":"TEST123"} and header x-ecpay-signature equal to HMAC-SHA256(rawBody, SECRET)
- Expected:
  - HTTP 200 OK
  - Logs contain digest of payload
